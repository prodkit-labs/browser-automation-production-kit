# Traffic Snapshot Template

Use this template to preserve short-window GitHub repository signals before
they disappear from repository insights.

This template is for open-source maintenance. It intentionally avoids private
site analytics, provider account tracking, and commercial tracking fields.

## Notes

- Record zeros or blanks instead of inventing numbers.
- Stars are useful trust signals, but referrers and popular content usually say
  more about user intent.
- Keep private promotion reasoning, provider account tracking, and service
  follow-up notes in a private control system, not in this public template.
