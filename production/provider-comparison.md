# Provider Comparison

This page will compare local execution, proxy-backed browser execution, and managed scraping/browser providers.

Evidence labels:

- `measured`: generated by benchmark scripts in this repo
- `estimated`: based on public pricing or documented assumptions
- `not tested`: included for future evaluation, not a recommendation

Required columns:

- provider
- execution mode
- success rate
- p95 latency
- cost per 1k pages
- block/captcha rate
- artifact support
- best fit
- tradeoffs

No provider should be described as the universal best option.
